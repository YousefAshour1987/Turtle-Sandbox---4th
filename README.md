# Turtle-Sandbox---4th
Draw the symbol found on Japanese Temples.  
In this assignment, you will build a simulation where turtles make the swastika. Note that:

1. Swastika ≠ Hakenkreuz
2. Swastika: It is an ancient symbol in the form of an equal-armed cross with each arm continued at a right angle, used especially in South Asia as a symbol of good luck or spirituality.
3. Hakenkreuz: A symbol in the form of a black cross with each arm bent clockwise at a right angle, used as the emblem of the German Nazi party.
